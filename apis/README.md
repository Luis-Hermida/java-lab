# APIs

## Description

This directory contains backend/API services, all written in Java.

## Examples

Suggested structure (per API service):

- `apis/<service-name>/` — individual API service
  - `src/main/java/...`
  - `src/test/java/...`
  - Build file (`build.gradle`, `pom.xml`, etc.)

We can standardize on Gradle or Maven later and add shared build conventions if needed.
